# Contributing

## Source of truth

All source files in `src/` are auto-synced from [`polyphon-ai/polyphon`](https://github.com/polyphon-ai/polyphon). Do not open pull requests here — they won't be accepted and the changes will be overwritten on the next sync.

To contribute to the SDK, open an issue or discussion in the main `polyphon` repository.

## Discussions

For questions, ideas, or feedback about the SDK, use [GitHub Discussions](https://github.com/polyphon-ai/.github/discussions).

## Security

Please do not report security vulnerabilities through public issues. See the [security policy](https://github.com/polyphon-ai/.github/blob/main/SECURITY.md) for details.
